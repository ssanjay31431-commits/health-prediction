# 🏥 AI Health Prediction System

An AI-powered Health Prediction System built using the MERN Stack.

## 🚀 Features

- 🔐 Secure Admin Login
- 👨‍⚕️ Super Admin Panel
- 👥 Patient Registration
- 🤖 AI Health Prediction
- 📊 Dashboard Analytics
- 📧 Automatic Email Report with PDF
- ⚠️ High Risk & Low Risk Detection
- 🔍 Search, Filter & Pagination
- 📱 Responsive Design

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- React Router

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

### Authentication
- JWT
- bcrypt

### Email
- Resend
- Requires `RESEND_API_KEY`, `RESEND_FROM_EMAIL`, and `RESEND_OWNER_EMAIL` in backend env for test-mode delivery

### PDF
- PDFKit

## 📂 Project Structure

```
backend/
frontend/
super-admin/
```

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Pavithramarriyappan/health-prediction-system.git
```

### Install Dependencies

```bash
cd backend
npm install

cd ../frontend
npm install

cd ../super-admin
npm install
```

### Start Backend

```bash
npm run dev
```

### Start Frontend

```bash
npm run dev
```
